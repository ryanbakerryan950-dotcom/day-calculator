# Kalkulator Hari

Kalkulator tanggal Indonesia — hitung selisih hari, tambah/kurang tanggal, hitung mundur, hari dalam seminggu, konversi Hijriah, dan Weton Jawa.

## Fitur

- **Selisih Hari** — Hitung jumlah hari antara dua tanggal (opsi hari kerja)
- **Tambah/Kurang** — Penambahan & pengurangan hari, minggu, bulan, atau tahun
- **Hitung Mundur** — Countdown real-time ke tanggal target
- **Hari dalam Seminggu** — Cari hari untuk tanggal tertentu
- **Konverter Hijriah** — Konversi dua arah Gregorian ↔ Hijriah
- **Weton Jawa** — Hitung Weton, Pasaran, dan Neptu

## Cara Menjalankan

Buka `index.html` langsung di browser, atau gunakan live server:

```bash
npx serve .
```
